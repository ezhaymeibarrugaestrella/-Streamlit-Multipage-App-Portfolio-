# 🌐 Streamlit Multipage Portfolio Web App

A multipage web application built using **Streamlit (Python)** featuring interactive pages such as Home, About, Projects, and Contact. The app showcases user-designed content, clean UI/UX, and dynamic components, including a personalized portfolio section.

---

## 📌 Features

- 🧭 Interactive pages (Home, About, Projects, Contact)
- 🎨 Clean and user-friendly UI/UX design
- ⚡ Dynamic components (buttons, inputs, real-time outputs)
- 👤 Personalized "About Me" page with profile image and custom styling
- 🖼️ Image handling using Base64 encoding
- 📁 Project showcase with descriptions and links
- 📱 Simple and responsive layout

---

## 🧑 About Page Highlights

- Displays profile picture in circular format
- Custom layout using HTML & CSS inside Streamlit
- Personal introduction and short bio
- Education and goals section
- Banner image for visual design
- Styled components for a modern portfolio feel

---

## 🛠️ Technologies Used

- Python 3
- Streamlit
- HTML & CSS (embedded in Streamlit)
- Base64 (for image rendering)
